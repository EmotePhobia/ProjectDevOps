# ProjectDevOps

## Instrukcja uruchomienia lokalnego


1. Sklonuj repozytorium: git clone https://github.com/EmotePhobia/ProjectDevOps.git
2. Zainstaluj zależności: npm install
3. Uruchom testy: npm test
4. Uruchom aplikację: npm start

### Jak uruchomić wersję LIVE?

1. Otwórz przeglądarkę internetową.
2. Wejdź pod adres:
   
   appkadevops-dke5emd0fwe7ftgq.polandcentral-01.azurewebsites.net

3. Aby sprawdzić endpoint z produktami, przejdź do:

   appkadevops-dke5emd0fwe7ftgq.polandcentral-01.azurewebsites.net/
   appkadevops-dke5emd0fwe7ftgq.polandcentral-01.azurewebsites.net/products
